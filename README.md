# Amplificador-de-Audio-multietapa-grupo-3
En este repositorio se encuentra el material documental del proyecto.

Simulación en Proteus e informe técnico de un amplificador de audio de tres etapas con transistores BJT.

## Archivos

PROYECTO_L2_DEFINITIVO.zip / Simulación completa en Proteus 
Informe_L2.pdf / Informe técnico con análisis y resultados 

## Cómo abrir la simulación

1. Descomprimir PROYECTO_L2_DEFINITIVO.zip
2. Abrir PROYECTO_L2_DEFINITIVO.pdsprj con **Proteus 8.x** o superior
3. Ejecutar con y verificar formas de onda en el osciloscopio virtual
